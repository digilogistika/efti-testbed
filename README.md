This is eFTI testbed - a playground for eFTI project member states.
This is a place for testing implementations of eFTI gate, platform and authority systems.

The eFTI TestBed is available here:
https://eu-ee32.eftisandbox.eu/

Contact [support@dlk.ee](mailto:support@dlk.ee) to get your gate/platform registered in the TestBed.

You will get a user, with which you can log in to the admin interface of your gate/platform.

Swagger / OpenAPI documentation is available for the different APIs:
https://eu-ee32.eftisandbox.eu/v1/openapi - eFTI API (Platform & Authority)
https://eu-ee32.eftisandbox.eu/api/openapi - Admin API
https://demo-platform.eu-ee32.eftisandbox.eu/v1/openapi - Platform own API
